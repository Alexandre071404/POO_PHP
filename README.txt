Ce projet à pour but des gérer une application en l'occurrence une liste d'animaux connus. 

Cela n'utilise pas de base de données en lignes mais une base de données locale dans laquelle nous auront des intéractions comme l'ajout d'un animal, sa suppression ou encore sa suppression. 

Ce projet est développé avec la volonté d'obtenir une application web utilisable en étant développé entièrement avec un modèle de programmation orienté objet Model/View/Controller.

Arborescence:

-Scr contient tous les fichiers nécéssaire au fonctionnement du site.

-Model contient tous les éléments objets qui seront utilisés dans le site.

-View contient tout le code qui affichera les différentes fonctionnalité du site.

-Control contient le fichier permettant de faire le lien entre le model et la View 

-Router php est le fichier permettant l'envoie et le traitement des divers requêtes de faites par l'utilisateur.

-Site.php est quant à lui le lanceur des divers fonctionnalité disponibles pour le site internet



The goal of this project is to manage an application, specifically a list of known animals.

It does not use an online database but rather a local database where interactions such as adding an animal, deleting an animal, or modifying an animal can take place.

This project is developed with the intention of creating a usable web application, fully developed using an Object-Oriented Programming Model/View/Controller (MVC) approach.

Structure:

-Scr contains all the necessary files for the site's operation.

-Model contains all the object elements that will be used on the site.

-View includes all the code that will display the various features of the site.

-Control contains the file that links the Model and the View.

-Router.php is the file that handles the sending and processing of the various requests made by the user.

-Site.php is the launcher for the various features available on the website.
